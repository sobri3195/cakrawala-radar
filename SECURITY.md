# Security Policy

Cakrawala Radar is a simulation, education, and research toolkit. It must not be used for weapon systems, targeting, interception guidance, evasion, jamming, spoofing, hacking radar systems, or accessing operational radar infrastructure.

## Supported Use

- Synthetic radar and ADS-B simulation.
- Academic anomaly detection and explainability.
- High-level airspace monitoring concepts.
- Safety-oriented decision support using synthetic or public user-supplied data.

## Reporting Issues

Please report security concerns through the project issue tracker or private maintainer contact once available. Include a minimal reproduction and avoid sharing sensitive data.

## Data Safety

The package does not perform automatic network calls. Users are responsible for ensuring any imported data is legal, public, synthetic, or properly authorized.
