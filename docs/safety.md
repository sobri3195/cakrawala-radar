# Safety

Cakrawala Radar is limited to simulation, education, academic research, anomaly detection, explainability, and high-level decision support using synthetic or public user-supplied data.

It must not be used for weapon systems, targeting, interception guidance, evasion, jamming, spoofing, radar hacking, unauthorized access to operational systems, tactical military instructions, or restricted data use.

The package does not perform automatic network calls.
